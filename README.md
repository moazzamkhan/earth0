# earth0
Earth0 - A different world where privacy exists.

# How to run

        // get source
        git clone https://github.com/moazzamkhan/earth0.git

        // move to project dir
        cd earth0/phi
        
        // install dependencies 
        yarn 
        OR 
        npm install

        // build code
        yarn build:electron

        OR

        npm run build:electron

        // start electron application 
        yarn start:electron
        
        OR
        
        npm run start:electron
